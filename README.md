# Students-performance-project-in-Python

📊 Students Performance Analysis
📌 Project Overview

This project explores a dataset about students’ academic performance to understand which factors most impact grades.

The goal is to apply exploratory data analysis (EDA) techniques in Python to identify patterns, correlations, and insights related to study habits and performance.

___

🎯 Key Questions

Do students who study more get higher grades?

Which habits most impact students’ performance?

Is there a strong correlation between study time and final score?

What is the standard deviation of grades?

____

🛠️ Tools & Technologies

Python

Pandas

NumPy

Matplotlib / Seaborn

Jupyter Notebook

___

📈 Key Insights

More studying generally leads to higher grades.

Performance variability (standard deviation) shows how spread out student results are.

Certain behavioral patterns strongly influence final outcomes.

Data cleaning

Exploratory Data Analysis (EDA)

Correlation analysis

Data visualization

Statistical concepts (mean, standard deviation)
